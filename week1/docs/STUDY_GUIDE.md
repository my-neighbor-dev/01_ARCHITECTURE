# 스터디 진행 가이드

## 스터디 전 준비

### 1. 질문 파일 공유
스터디 시작 전에 `00-questions.md` 파일을 공유

### 2. 프로젝트 준비
- 프로젝트 클론 또는 다운로드
- IDE에서 프로젝트 열기
- 빌드 확인: `./gradlew build`

---

## 추가 자료

- `PROJECT_STRUCTURE.md`: 프로젝트 구조 상세 설명
- `README.md`: 프로젝트 개요 및 실행 방법

## 스터디 후 과제 (선택사항)

1. 자신의 프로젝트에 적용해보기
2. Before/After 비교 문서 작성
3. 팀 내 공유 및 리뷰
